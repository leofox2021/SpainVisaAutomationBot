Боту требуется последняя версия Google Chrome (на данный момент - 103).
Если у вас нету данного браузера, или стоит Chrome 102 и ниже, открываем файл ChromeSetup.exe и обновляемся.
Альтернативный вариант - в Google Chrome открываем настройки - help - about Google chrome и обновляемся там. 

Версии программы:
bot_demo.exe - демо версия версия программы.

Папка DATA
Папка содержит информаицю о для заполнения ей других файлов из папки CONTEXT.
passport_types.txt - типы паспортов представленные на сайте. 
visa_centers.txt - все доступные города для подачи визы представленные на сайте.
visa_types.txt - типы виз представленные на сайте.
visa_hours.txt - возможные часы посещения визового центра представленные на сайте.

Папка CONTEXT
Папака содержит текстовые файлы с заполняемой пользователем информацией:
phone.txt - заполните свой номер телефона без кода региона (+7).
travel_date.txt - заполните планируемую дату поездки в формате ДД ММ ГГГГ без дополнительных символов или пробелов
(пример- 20102022).
visa_center.txt - заполните город посещения (скопируйте из 'data/visa_centers').

Вырезанные функции: 
email.txt - введите свой email чтобы автоматически заполнить код подтверждения. 
Доступно в полной версии. 
members.txt - укажите количество человек для подачи. На данный момент не функционирует должным образом. 
Из-за ошибок на сайте откатывает обратно к 2 пользователям за один раз. 
booking_date.txt - укажите дату для посещения визового центра. Недоступно из-за ошибок на сайте, 
приводящих к вылету обратно на экран регистрации.  

Таблица USER_DATA
Заносить не более 2 человек (больше не поддерживает сайт, возможно в скором времени будет исправлено).
Заполнять таблицу строго по указанным шаблонам - синтаксическая ошибка приведет к остановке работы бота. 
Название менять нельзя, так как и переносить в другую папку, потому что бот перестанет ее идентифицировать.
Если номер в таблице начинается с нуля, перед нулем нужно поставить апостроф: '05092025
При заполнении времени посещения визового центра, некоторые редакторы таблиц могуть менять маленький знек "-" на длинный. 
При случае этого необходимо будет исправить знак.
Как можно: 14:15 - 14:30
Как нельзя: 14:15 – 14:30

Время посещения: скопировать информацию из 'data/visit_hours.txt' (пример: 11:00 - 11:15).
Тип паспорта: скопировать информаицю из 'data/passport_types.txt' (пример: Заграничный паспорт).
Тип визы: скопировать информаицю из 'data/visa_types.txt' (пример: 5 лет - Шенгенская виза С).
Номер паспорта: ввести номер заграничного паспорта без пробелов и других знаков. 
Имя: ввести имя заглавными латинскими буквами.
Фамилия: ввести фамилию заглавными латинскими буквами.
День рождения: заполните день рождения в формате ДД ММ ГГГГ без дополнительных символов или пробелов (пример - 20101986).
Дата выдачи: заполните дату выдачи заграничного паспорта в формате ДД ММ ГГГГ без дополнительных символов или пробелов (пример - 20102018).
Дата окончания: заполните дату окончания действия заграничного паспорта в формате ДД ММ ГГГГ без дополнительных символов или пробелов (пример - 19072030).
Место выдачи: заполните город выдачи заграничного паспорта.
